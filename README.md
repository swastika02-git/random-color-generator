# random-color-generator
A simple web application built using HTML, CSS, and JavaScript that generates a random hexadecimal color on each button click. The generated color code is displayed on the screen, and the webpage background updates to match the selected color.
# 🎨 Random Color Generator

A simple web application built using **HTML, CSS, and JavaScript** that generates a random hexadecimal color each time the button is clicked. The generated color code is displayed on the screen, and the webpage background changes to the generated color.

## 🚀 Features

- Generate a random hex color with a single click
- Display the generated color code
- Automatically change the webpage background
- Simple and responsive user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
random-color-generator/
│── index.html
│── style.css
│── script.js
└── README.md
```
